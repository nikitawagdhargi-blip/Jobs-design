Spacing Scale

Strict 5-value scale. No random increments. Every value serves a purpose.

Name Value CSS Variable Usage
Extra Small 8px --space-xs Tight spacing, adjacent elements
Small 16px --space-sm Component margins, form helpers
Medium 24px --space-md Standard padding, gaps (most common)
Large 40px --space-lg Major sections, page padding
Extra Large 64px --space-xl Hero areas, major visual breaks
